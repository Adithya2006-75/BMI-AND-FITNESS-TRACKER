# BMI-AND-FITNESS-TRACKER
FitForge – Ultimate Fitness Tracker
FitForge is a modern, feature-rich fitness tracking web application designed to help users monitor workouts, track progress, and maintain consistency in their fitness journey. It provides an all-in-one solution including workout logging, timers, analytics, streak tracking, BMI calculation, and more.

Features
1.Workout Logging
  Log exercises based on muscle groups
  Add sets with repetitions and weight
  Automatically calculates total sets and total volume
  Stores daily workout data locally
2.Timer System
  Stopwatch for tracking workout duration
  Rest timer with preset options and manual adjustments
  Lap tracking functionality
3.Progress Tracking
  Weekly volume tracking
  Daily sets tracking
  Category-based workout distribution
  Workout frequency analysis
  Charts powered by Chart.js
4.Streak Tracking
  Tracks daily workout consistency
  Calculates best streak
  Displays weekly and monthly calendars
  Visual indicators for workout completion
5.Weekly Timetable
  Plan workouts for each day of the week
  Add and remove exercises with sets and reps
  Structured weekly scheduling
6.BMI Calculator
  Supports metric and imperial units
  Displays BMI value and category
  Includes a visual indicator scale
7.Personal Records
  Automatically detects new personal records
  Stores highest weight lifted per exercise
  Displays ranked records
8.User Interface
  Clean and modern dark theme
  Mobile-friendly responsive design
  Smooth navigation with a bottom navigation bar

Technologies Used
  HTML5
  CSS3
  JavaScript (Vanilla)
  Chart.js
  LocalStorage API

How It Works
  All data is stored in the browser using LocalStorage
  No backend or database is required
  Fully client-side application
  Works offline after loading
  
How to Run
  1.Download or clone the project
  2.Open the index.html file in a browser
  3.Start using the application
